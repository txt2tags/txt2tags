About txt2tags

Txt2tags is a document generator. It reads a text file with minimal markup such as **bold** and //italic// and converts it to the following formats:

  • HTML
  • XHTML
  • SGML
  • DocBook (NEW)
  • LaTeX
  • Lout
  • Man page
  • Creole (NEW)
  • Wikipedia / MediaWiki
  • Google Code Wiki
  • PmWiki (NEW)
  • DokuWiki
  • MoinMoin
  • MagicPoint
  • PageMaker
  • AsciiDoc (NEW)
  • ASCII Art (NEW)
  • Plain text

About this site

This site is 100% powered by txt2tags and CSS. No Javascript, CGI or PHP, just plain static valid HTML pages.

You can read the text sources for each page, just click the "See sources" link on the footer. Reading them you will learn the markup instantly.

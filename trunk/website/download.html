Download — All platforms

  • txt2tags-2.6.tgz — 545 KB, released in 05 Nov 2010

Windows

  • Install Python and download txt2tags

Mac OS X

  • txt2tags-2.6-0.dmg

  • MacPorts: sudo port install txt2tags
  • homebrew: brew install txt2tags

Linux

Arch: sudo pacman -S txt2tags +info
CentOS: yum install txt2tags RPM package
Debian: sudo apt-get install txt2tags +info
Fedora: yum install txt2tags
Gentoo: emerge txt2tags +info
OpenSUSE: Add the Application:Geo repository
Red Hat: yum install txt2tags RPM package
Slackware: Download tgz
Ubuntu: sudo apt-get install txt2tags +info

BSD

  • FreeBSD: http://www.freshports.org/textproc/txt2tags/
  • NetBSD: http://pkgsrc.se/wip/txt2tags
  • OpenBSD: http://ports.openbsd.nu/textproc/txt2tags


Notes:

  • Requires Python 2 (works from 2.2 to 2.7). It won't work in Python 3 UPDATE: @jbvsmo ported the code to work in Python 3, try it out!

  • This release news are detailed in the blog announce.

  • Read the full list of changes since the last version.

  • See some installation tips from the User Guide.

  • New to txt2tags? Read this, this and this.

  • You can also download just the Python script, if you know what you're doing.

  • Python developers can also install txt2tags as a Python module:
    easy_install -f "http://txt2tags.org/txt2tags.py#egg=txt2tags-2.6" txt2tags

  • If you have a feature request or found a bug, please submit an issue.

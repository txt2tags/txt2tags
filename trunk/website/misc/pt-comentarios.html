Quem costuma manter suas anotações em arquivos de texto puro, geralmente o faz porque isso evita preocupações supérfluas com a apresentação do texto, o que ajuda a manter o foco em seu conteúdo. Os problemas começam a surgir quando aparece a necessidade de imprimir o texto ou divulgá-lo num formato mais apresentável. Como conheceu o txt2tags

Eu estava lendo sobre sed no site do Aurélio e para descontrair decidi testar alguns dos scripts que haviam por lá... adorei as funções zz.

Porque ele é útil para você

Sou programador e gerencio a área de TI da AmbiTerra. Lá nós desenvolvemos muitas ferramentas para a empresa mas criar documentação tanto para o usuário como para nós mesmos sempre foi um problema, temos muitas coisas para fazer e não dá para ficar perdendo tempo com formatação de documentos e nossa metodologia de programação gera pouca doucumentação (eXtreme Programming). Nossos documentos têm que reunir as seguintes características:

Portabilidade, temos usuários lendo esses documentos em todo tipo de plataforma, de handhelds a PCs e não dá para prever qual sistema operacional está instalado ou se tem um pacote de aplicativos como o Office por exemplo.

Facil manutenção, sem tags complicadas ou editores estranhos, foco no conteúdo de forma que com pouco treinamento qualquer pessoa possa atualizar um documento.

Arquivo fonte em "texto puro" para que possamos usar coisas úteis como sed e manter tudo no CVS.

Layout automático, todos os títulos, subtítulos, margens, etc, devem estar com tamanho correto, fonte correta, etc.. sem que tenhamos que fazer nada ou quase nada.

A idéia inicial era escrever tudo em HTML mas as tags são muitas e são complicadas, os usuários têm uma tendência a usar editores WYSIWYG que enchem o arquivo de lixo tornando impossível dar manutenção sem continuar usando esses editores. Nós queriamos algo que pudéssemos abrir no VIM e entender o conteúdo de imediatamente sem ter que ficar procurando o texto no meio das tags.

Quais as melhores características do programa

  1. Tags descomplicadas, arquivo fonte legível.
  2. Gerar saída em múltiplos formatos.
  3. É um software livre.

Como conheceu o txt2tags

É uma longa história. Após ler um artigo do Aurélio na RdL enviei um email perguntando e recebi a resposta. Fiquei conhecendo o 'Guia de Expressões Regulares'. Gostei e fui no site do autor. Foi o meu primeiro contato com o txt2tags. Uma decepção.

Um shell script que gerava um programa em Python para gerar a página, etc. Sem muita utilidade para um usuário do Windows (baixar o Cygwin e Python só para isto). Olhei outro programas que geravam páginas html partindo de texto mas nenhum me agradou muito. Comecei a desenvolver o meu em Ruby.

Tive que interrompê-lo e após algum tempo o txt2tags já havia evoluído. Podia ser utilizado no Windows, com uma interface gráfica (tudo que um usuário do Windows quer :) e precisava apenas do Python (coisa que eu já possuia) E cá estou eu, utilizando o txt2tags, procurando divulgar o mesmo e contribuindo de uma ou outra forma.

Porque ele é útil para você

Por permitir que eu gere uma página html tendo como fonte um texto simples com algumas marcações simples e me dando a segurança de poder gerar em outros formatos se eu precisar, utilizando o mesmo fonte.

Quais as melhores características do programa

  1. Permitir, com alguns parâmetros, alterações na formatação do documento final.
  2. Comando para um ajuste ajuste fino no pré e pós processamento.
  3. Possibilidade de alterações no código fonte para atender a necessidades especifica (se precisar)

Documentos convertidos com ele

http://ruby-ptbr.rubyforge.org/

Comentário final

Apesar de existirem outras maneiras de gerar documentos, o txt2tags me pareceu uma das mais simples porém poderosa ferramenta do gênero. Após conhecer alguns marcadores, qualquer um está apto a ser (pensar que é) webdesigner :)

Espero que continue em evolução mantendo o princípio bem como aumente a legião de usuários. Apesar de não ser autor, fico contente quando vejo um projeto dando certo, crescendo, sendo comentado nas diversas rodas. Ainda mais sendo de origem nacional e mantido por um cara que, mesmo não conhecendo pessoalmente, já considero como amigo. Valeu por compartilhar o programa com todos nós.

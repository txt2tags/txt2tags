Вступ

Вітаємо у файлі прикладі txt2tags.

Тут ви отримаєте приклади та короткі описи всіх позначень розмітки. marks.

Перші три рядки цього файлу використовуються як заголовки, в такому форматі: 

  рядок1: заголовок документа
  рядок2: ім'я та контакти автора
  рядок3: дата, версія

Рядки з однаковою кількістю знаків дорівнює по краях - заголовки.

Шрифти та виділення

Ми маємо два види шрифтів:

ЗВИЧАЙНИЙ, який може змінюватись різними виділеннями.

Та ДРУКАРСЬКИЙ моноширинний для попередньо відформатованого тексту.

А тепер перейдемо до підрозділу...

Виділення

Текстові позначки для виділень прості, й ви їх можливо вже використовували в переписці електронною поштою.

Ми використовуємо подвійні *, /, - та _ щоб передати жирний, курсивний, закреслений та підкреслений шрифти.

Підтримується також жирний курсив як комбінація двох.

Попередньо відформатований текст

Ми можемо вставляти шматки коду, чи інший попередньо відформатований текст: 

    ось   попередньо віформатований  текст
  //позначення// розмітки  **не**  ``інтерпретуються``

Також дуже просто вставити однорядковий код: 

  prompt$ ls /etc

Чи використати такий шрифт всередині речення.

Більше оформлення

Спеціальні елементи, такі як email (duh@somewhere.com) чи URL (http://www.duh.com) визначаються автоматично, як і горизонтальні лінії:

^ тонкі чи товсті v

Можна також задати посилання з іменем.

І пам'ятайте,

Табуляція на початку рядка позначає цитату.
Чим більше табів, тим більша глибина вкладень (якщо дозволяє цільова мова).

Гарно.

Списки

Список створюється очевидно, додайте дефіс чи плюс на початку рядка.

Ненумерований список

Дефіс (мінус) є основним ідентифікатором списку. Для створення підсписку просто додайте прогаликів на початку рядка. Чим більше прогаликів, тим більша глибина вкладення.

  • Земля
    • Європа
      • Східна Європа
        • Україна
          • Як довго можна уточнювати?
    • Азія
      • Великі країни
  • Марс
    • Хто зна?

Список закінчують двома послідовними порожніми рядками.

Нумерований список

Такі самі правила як і для ненумерованого, лиш інше позначення (плюс).

  1. раз
  2. два
  3. три
    • змішані списки!
    • яка плутанина
      1. рахуємо знову
      2. ...
  4. чотири

Список означень

Список означень це двокрапка, після якої пишуть термін. Означення терміну розміщують у наступному рядку.

помаранча
помаранчевий фрукт
яблуко
зелений або червоний фрукт
Інші фрукти
  • Єй!
  • Змішані списки
    1. знову!
    2. і знову!

Таблиці

Використовуйте символ вертикальної риски (трубу), щоб оформлювати таблиці. Дві труби на початку рядка позначають заголовочний рядок. Пропуски задають вирівнювання всередині таблиці.

heading 1 heading 2 heading 3
cell 1.1 cell 1.2 cell 1.3
cell 2.1 cell 2.2 cell 2.3

Без останньої риски межі будуть невидимими:

heading 1 heading 2 heading 3
cell 1.1 cell 1.2 cell 1.3
cell 2.1 cell 2.2 cell 2.3

Зображення

Простіше позначити зображення неможливо: [filename].

  • Назва файлу має закінчуватись на PNG, JPG, GIF, чи подібне.
  • Ніяких пропусків всередині квадратних дужок!

Інше

Зручний макрос %%date замінюється на поточну дату.

Наприклад зараз 20100804 в форматі ISO YYYYMMDD.

Також можна задати формат дати символами %?, як наприклад %%date(%m-%d-%Y) дасть: 08-04-2010.
